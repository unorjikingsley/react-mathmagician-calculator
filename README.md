## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- Copy [link] (https://github.com/unorjikingsley/react-mathmagician-calculator.git)

### Setup

Clone this repository to your desired folder:

- Open terminal or command prompt and run git clone https://github.com/unorjikingsley/react-mathmagician-calculator.git

### Install

Install this project with:

- cd react-mathmagician-calculator
- run `npm i`

### Usage

To run the project, execute the following command:

- Run `npm run start`
